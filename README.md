## 开始使用

1. 添加 css
    使用本框架前，请在 css 中开启 border-box
    ```
    *,*::before,*::after{box-sizing:border-box}
    ```
2. 安装

    ```
    npm install milk-ui-test
    ```

3. 引入 milk-ui-test

    ```
    import {Button} from 'milk-ui-test'
    import 'milk-ui-test/dist/index.css'
    export default {
    components: {
        'm-button':Button
    }
    }
    ```
